Пример модуля модальных окон

  • Первый пример

    • Инициализация модального окна с контентом по умолчанию.
    • Вставка нового контента в модальное окно через определенное время.

  • Второй пример

    • Инициализация модального окна.
    • Вставка контента в модальное окно.
    • Удаление модального окна из дерева dom.
    • Реинициализация модального окна.
    • Вставка нового контента в модальное окно через определенное время.

  • Третий пример

    • Инициализация модального окна через метод Popup.makePopup.
    • Вставка компонента Vue в модальное окно.

  • Четвертый пример

    • Инициализация модального окна через метод Popup.makeSinglePopup, возвращающий singleton-экземпляр модального окна.
    • Вставка компонента Vue в модальное окно.
    • Вставка компонента Vue в футер модального окна.

  • Пятый пример

    • Открываем несколько модальных окон.
    • Закрываем их все через определенное время.

  • Шестой пример

    • Открываем пользовательский компонент.
    • Присутствует динамический контент.

  • Седьмой пример

    • Создаем модальное окно через метод Popup.makeDraggableModal.
    • Открываем модальное окно.
    • Можно подвигать модальное окно.

  • Восьмой пример

    • Создаем модальное окно.
    • Открываем модальное окно.
    • Показываем в модальном окне поочередно элементы с определенным контентом
